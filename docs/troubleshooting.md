# Troubleshooting Guide

### ✔ Excel not loading?
- Make sure the file is stored in **OneDrive** or **SharePoint**, not local PC.
- Ensure table formatting is applied: **Insert → Table**.

### ✔ Flow fails in loop?
- Check that required fields (task title, due date) are not empty.

### ✔ Planner task not created?
- Verify you have access to the Planner group.
- Check that the Plan ID and Bucket ID are correct.

### ✔ Reminders not sending?
- Verify email or Teams connector has permissions.
