# FAQ — Planner Automation Support

### ❓ Do I need a premium license?
No — all flows provided here work with standard Microsoft 365 licenses.

### ❓ Can I add custom fields?
Yes, use the "Update Task" action and modify Excel columns.

### ❓ Can I run flows manually?
Yes, all flows support **manual trigger** + automated triggers.

### ❓ Does this work for Planner Basic Plans?
Yes, the flows are designed for Planner Basic (non-Premium) plans.
